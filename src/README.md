# Project Code

The figures of the paper can be generated by running the code in the jupyter notebook `generate_figures.ipynb`. This notebook has the following python packages as dependencies:

- `numpy`
- `scipy`
- `matplotlib`

The other files in this directory contain definitions and helper functions:

- `chord.py` - functions to compute distances on a circle 
- `coherence.py` - functions to compute the two-RDM coherences
- `energy.py` - function to compute the lattice model energiy estimates from the two-RDM
- `LL_definitions.py` - functions to compute Luttinger liquid parameters and relate them to $J$-$V$ model parameters
- `load.py` - functions to load the DMRG data
- `one-rdm.py` - functions to compute the one-RDM 
- `structurefactor.py` - functions to compute the structure factor 
- `two-rdm.py` - functions to compute the two-RDM
